  Visual Search allows you to find near-duplicates, repeated patterns, and conceptually similar content in your dataset—whether at the image or object level. It’s ideal for identifying anomalies, clustering related visuals, and accelerating QA review.
</Card>

## Overview

Visual Layer enables intuitive similarity search across datasets using deep visual embeddings. You can run a similarity search in several ways:

- [On a cluster](#similarity-search-on-a-cluster)  
- [On an image or object](#similarity-search-on-an-image-or-object)  
- [Using a region of interest](#similarity-search-using-a-region-of-interest)  
- [Via image upload](#similarity-search-via-image-upload)

## Similarity Search on a Cluster

To find similar results based on a full cluster:

1. Hover over the desired cluster in your dataset.
2. Click **Find Similar** at the bottom of the card.

## Similarity Search on an Image or Object

You can also trigger similarity search directly from any image or object. There are multiple entry points depending on your view:

### Search from the Dataset Page

1. Click into a cluster.
2. Hover over the image or object and click **Find Similar**.

### Search from the Cluster Details Page

1. Click an image or object to open the **Image Details Page**.
2. Click **Find Similar**, then click the similarity icon centered on the image.

### Search by Object

To run similarity search based on a specific object in the image:

1. Hover over the object’s bounding box.
2. Click the **Find Similar** icon.

## Similarity Search Using a Region of Interest

You can crop part of an image and use that region as a visual anchor:

1. Open an image from any cluster.
2. Click **Find Similar**.
3. Drag to define the Region of Interest.
4. Click the similarity icon again to start the search.

## Similarity Search via Image Upload

You can also start a similarity search using an external image:

1. On the Dataset Page, click the **Find Similar** icon near the search bar.
2. Upload an image from your local device.
3. Optionally crop it before starting the search.
