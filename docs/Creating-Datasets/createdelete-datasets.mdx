title: "Upload data"
description: "Learn how to create datasets with Visual Layer."
---

Create datasets in Visual Layer by uploading your own files, folders, or linking to external storage sources. Follow the steps below to configure your dataset, preview your upload, and (optionally) add annotations.

## 1. Create a New Dataset

To get started:

1. Navigate to the **Dataset Inventory** page.
2. Click **+** icon or **New Dataset**.

![Create a new dataset](https://files.readme.io/ca369b8add29da612e26061c671355ac052b2aae43419f79ba9432c80f001eb4-image.png)

---

### Name Your Dataset

You can enter a custom name or use the name auto-suggested by Visual Layer.

<Info>
Avoid using duplicate names.
</Info>

![Name your dataset](https://files.readme.io/4ab83c6e298ec22e1358a479bf4faa1115607dbafed628ba1b621fa76f4b25c4-image.png)

---

### Select Data Source

Choose where to upload your data from. Visual Layer supports the following data source types:

- ZIP file  
- Files or Folders  
- Public Amazon S3  
- Private Amazon S3

![Select data source](https://files.readme.io/3484a6682d53ec4fc182aa1d83deac9247e820deecd311a38e22c74c2b4f2ae3-image.png)

<Tip>
Need another data source? Contact [support@visual-layer.com](mailto:support@visual-layer.com) or use the **Contact Us** form.
</Tip>

### Preview Your Data

Once uploaded, Visual Layer generates a preview so you can validate your files.

- To replace the selected source, click **Edit** and follow the prompts.

![Data preview](https://files.readme.io/ca8f63b185bf1a6e7cd519f2f3142444a250f57a90a36fd99e987d13655e6c73-image.png)


## 2. Add Annotations (Optional)

You can upload an annotation file at this stage to enrich your dataset.

1. Upload the annotation file and click **Next**.
2. A new preview will show the dataset with annotations applied.
3. Click **Edit** if you need to replace the file.

![Annotation preview](https://files.readme.io/9a0e4cd3f9347a88f1e04cdc06e811550cc7ac88d39d672fe38f5a9c4c21f806-image.png)

> Skip this step if you don't have annotation files.  
> For supported formats, see [Upload Annotation](https://dash.readme.com/project/visual-layer2/v1.0/docs/importing-annotations-copy).


## 3. Upload and Index

After confirming your data and annotations (if applicable):

- Click **Next** to start the upload and indexing process.
- Uploads are subject to your current plan’s limits.
- You’ll receive an email when the dataset is ready.

## 4. Delete a Dataset

To remove an existing dataset:

1. Go to **Dataset Inventory**.
2. Click the menu next to the dataset.
3. Select **Delete**.

<Info>
Deleting a dataset is irreversible.
</Info>

![Delete dataset](https://files.readme.io/c1f37f6fa067084f90efe3253ca59f07d9f2466593fa834e5a6b5d89adc95045-image.png)
